## 📝 文档命名规范

所有文档遵循以下命名规范：

```
<日期YYYYMMDD>-<文档名称>.md
```

例如：
- `20260308-MaiBot新架构使用指南.md`
- `20260227-GitHub推送指南.md`

**注意：** 只有 `README.md` 不带日期前缀。
