## 📖 使用指南

### 查找文档

1. **查看文档索引**
   ```
   打开 docs/README.md
   ```

2. **按类型查找**
   - 架构设计 → `docs/architecture/`
   - 使用指南 → `docs/guides/`
   - 归档文档 → `docs/archive/`

3. **快速导航**
   - 在 `docs/README.md` 中使用"快速导航"部分

### 添加新文档

1. **确定文档类型**
   - 架构设计 → `architecture/`
   - 使用指南 → `guides/`
   - 归档文档 → `archive/`

2. **命名文档**
   ```
   <文档名称>-<YYYYMMDD>.md
   ```

3. **更新索引**
   - 在 `docs/README.md` 中添加链接
   - 更新"文档更新记录"

### 归档旧文档

1. **移动到 archive/**
   ```bash
   mv docs/guides/旧文档.md docs/archive/旧文档-YYYYMMDD.md
   ```

2. **更新索引**
   - 从对应分类中移除
   - 添加到归档分类
