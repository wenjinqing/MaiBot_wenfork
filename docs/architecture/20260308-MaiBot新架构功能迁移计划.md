# MaiBot 新架构功能迁移计划

> 创建日期：2026-03-08
> 状态：规划中

## 📋 概述

将旧架构中的关系系统、心情系统、表情包系统、记忆系统迁移到新架构（chat_v2）中。

---

## 🎯 阶段 2：功能迁移

### 任务 1：迁移关系系统 ⏳

**目标：** 将关系系统集成到新架构的 UnifiedAgent 中

**涉及文件：**
- `src/common/relationship_updater.py` - 关系更新器
- `src/common/relationship_query.py` - 关系查询
- `src/common/relationship_history_manager.py` - 关系历史管理
- `src/common/relationship_penalty.py` - 关系惩罚

**集成点：**
1. 在 `_build_context()` 中添加用户关系信息
2. 在 `_build_system_prompt()` 中包含关系等级
3. 在消息处理完成后更新关系值

**实现步骤：**
1. [ ] 在 AgentContext 中添加 relationship_info 字段
2. [ ] 在 UnifiedAgent._build_context() 中查询关系信息
3. [ ] 在 UnifiedAgent._build_system_prompt() 中包含关系信息
4. [ ] 在 UnifiedAgent.process() 结束时更新关系值
5. [ ] 测试关系系统是否正常工作
