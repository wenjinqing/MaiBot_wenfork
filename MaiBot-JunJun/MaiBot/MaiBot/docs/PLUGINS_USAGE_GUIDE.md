### 1.4 新闻功能

**触发方式：** 关键词触发、LLM工具调用或命令

**关键词触发：**
- `今日新闻` - 获取今日60秒新闻
- `今天新闻` - 获取今日60秒新闻
- `新闻` - 获取今日60秒新闻
- `60秒新闻` - 获取今日60秒新闻
- `历史上的今天` - 获取历史上的今天事件
- `今天的历史` - 获取历史上的今天事件
- `历史事件` - 获取历史上的今天事件

**命令：**
- `/news` 或 `/新闻` - 获取今日60秒新闻
- `/history` 或 `/历史` - 获取历史上的今天

**使用方法：**
```
今日新闻
历史上的今天
/news
/history
```

**说明：** 
- 关键词触发：直接发送关键词即可（如"今日新闻"、"历史上的今天"）
- 命令触发：使用 `/news` 或 `/history` 命令
- `/news` 获取今日热点新闻（10-15条+微语）
- `/history` 获取历史上的今天事件

### 1.6 AI绘图功能

**触发方式：** 命令触发或LLM工具调用

**命令：**
- `/draw <描述词>` - AI绘图
- `/绘图 <描述词>` - AI绘图
- `/画图 <描述词>` - AI绘图

**使用方法：**
```
/draw 猫娘
/绘图 可爱的猫咪
/画图 动漫少女
```

**特殊用法：**
- `/draw self` 或 `/draw 你自己` - 根据bot人设生成图片
- 用户说"换个风格"、"再来一张" - 切换图片风格

**说明：**
- ⚠️ **仅在用户明确要求画图时才会调用**
- 不要自动配图，不要为了增强表现力而自动调用
- 支持智能风格匹配和图片缓存

## 2. 抽老婆插件 (wife_plugin)

**触发方式：** 关键词触发（仅群聊）

**关键词：**
- `抽老婆`
- `今日老婆`
- `谁是我老婆`

**使用方法：**
```
抽老婆
今日老婆
谁是我老婆
```

**功能说明：**
- 在群聊中随机选择一位群成员作为"群老婆"
- 每天每个用户只能抽一次
- 如果今天已经抽过，使用"今日老婆"或"谁是我老婆"可以查看已抽取的结果
- 如果抽到麦麦（bot），会有特殊提示

**配置说明：**
- 需要在 Napcat WebUI 中配置独立的 HTTP 服务器（端口6666）
- 详细配置请参考 `plugins/wife_plugin/docs/README.md`

**黑幕功能：**
- 在配置文件中设置 `target` 为某个QQ号，该用户每天必定抽到麦麦

## 3. Google 搜索插件 (google_search_plugin)

**触发方式：** LLM工具自动调用（需在 `bot_config.toml` 中启用工具调用）

**前置条件：**
- 在 `bot_config.toml` 中设置 `[tool] enable_tool = true`
- 已安装所有依赖（运行 `pip install -r requirements.txt`）

**使用方法：**
```
能搜一下最近很火的《Ave Mujica》吗？
找一张千早爱音图片给我~
```

**功能说明：**
- 当用户提出需要外部知识或最新信息的问题时，插件会自动被触发
- 支持多个搜索引擎：Google、Bing、搜狗、DuckDuckGo、Tavily
- 自动查询重写、内容抓取、智能总结

**支持的搜索引擎：**
- Google（需配置代理）
- Bing（默认）
- 搜狗
- DuckDuckGo
- Tavily（需API key）

**图片搜索：**
- 需要在配置中启用 `actions.image_search.enabled = true`
- 需要配置代理（搜图功能无法直连）
- 用户明确要求搜索图片时才会调用

**缩写翻译：**
- 自动识别网络用语、缩写、黑话并翻译
- 例如：yyds、u1s1、躺平、内卷等

## 4. 聊天频率控制插件 (ChatFrequency)

**触发方式：** 命令触发

**命令：**
- `/chat talk_frequency <数字>` 或 `/chat t <数字>` - 设置当前聊天的发言频率调整值
- `/chat show` 或 `/chat s` - 显示当前聊天的频率控制状态

**使用方法：**
```
/chat t 0.5
/chat talk_frequency 1.5
/chat show
/chat s
```

**功能说明：**
- 调整当前聊天的发言频率（talk_frequency）
- 可以增加或减少麦麦在该聊天的发言频率
- 支持正数和负数（负数会降低发言频率）

**示例：**
- `/chat t 0.5` - 将发言频率调整为原来的0.5倍（降低发言频率）
- `/chat t 1.5` - 将发言频率调整为原来的1.5倍（提高发言频率）
- `/chat t -0.3` - 降低发言频率0.3

## 📝 使用提示

1. **命令格式：** 所有命令都以 `/` 开头
2. **关键词触发：** 部分功能通过关键词自动触发，无需命令
3. **工具调用：** 部分功能（如搜索）需要LLM工具调用，需在 `bot_config.toml` 中启用
4. **群聊/私聊：** 部分功能仅在群聊或私聊中可用，请查看具体说明
5. **配置要求：** 部分插件需要额外配置（如API密钥、代理等），请查看插件文档

---

## 🔧 配置检查

### 工具调用配置
确保 `MaiBot/config/bot_config.toml` 中：
```toml
[tool]
enable_tool = true  # 启用工具调用（搜索插件需要）
```

### 插件启用状态
各插件的启用状态可在对应的 `config.toml` 文件中查看和修改。
