## 常见问题

### Q: 推送失败，提示文件太大？

A: 检查是否包含了大文件：
```bash
git ls-files | xargs ls -lh | sort -k5 -hr | head -20
```

如果有大文件，添加到 .gitignore 并重新提交。

### Q: 推送很慢？

A: 1701 个文件第一次推送会比较慢，这是正常的。后续推送会快很多。

### Q: 如何撤销提交？

A: 如果发现包含了敏感信息：
```bash
# 撤销最后一次提交
git reset --soft HEAD~1

# 修改 .gitignore
# 重新添加文件
git add .

# 重新提交
git commit -m "..."
```
