## 🔍 故障排查

### 问题 1：日志中没有 "使用新架构"

**原因：** 配置未生效

**解决：**
1. 确认 `bot_config.toml` 中 `use_v2_architecture = true`
2. 确认文件已保存
3. 重启程序

### 问题 2：启动报错

**可能的错误：**
```
ModuleNotFoundError: No module named 'src.chat_v2'
```

**解决：**
1. 确认 `src/chat_v2/` 目录存在
2. 确认所有 `__init__.py` 文件存在
3. 检查 Python 路径

### 问题 3：工具调用失败

**查看日志：**
```
[tool_executor_v2] 工具 web_search 执行失败: ...
```

**解决：**
1. 检查工具是否正确注册
2. 检查网络连接
3. 查看详细错误信息

### 问题 4：回复质量下降

**可能原因：**
- 提示词需要调整
- 模型配置需要优化

**解决：**
1. 查看 `src/chat_v2/agent/unified_agent.py`
2. 调整 `_build_system_prompt` 方法
3. 修改 temperature 参数

## 📈 优化建议

### 1. 调整工具超时时间

编辑 `src/chat_v2/executor/tool_executor.py`：

```python
async def execute_tools(self, tool_calls, timeout=30.0):  # 默认 30 秒
    ...
```

根据实际情况调整 `timeout` 参数。

### 2. 调整 LLM 参数

编辑 `src/chat_v2/agent/unified_agent.py`：

```python
success, response, ... = await llm_api.generate_with_model_with_tools(
    ...
    temperature=0.8,  # 调整这里
    ...
)
```

- `temperature` 越高，回复越随机
- `temperature` 越低，回复越确定

### 3. 优化提示词

编辑 `src/chat_v2/agent/unified_agent.py` 中的 `_build_system_prompt` 方法。
