## 📝 目录维护建议

### 定期清理（推荐）

**每周清理**:
```bash
# 清理测试缓存
rm -rf htmlcov/ .pytest_cache/

# 清理 Python 缓存
find . -type d -name "__pycache__" -exec rm -rf {} +
```

**每月清理**:
```bash
# 清理30天前的日志
find logs/ -name "*.log*" -mtime +30 -delete
```

**按需清理**:
```bash
# 完全清理虚拟环境（需要重新安装依赖）
rm -rf .venv/ venv/

# 清理数据库（谨慎！会丢失所有数据）
rm -rf data/
```
