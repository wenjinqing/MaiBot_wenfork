# 快速启动 - 3步搞定

## 第1步: 配置白名单（首次必须）

### 君君的白名单
编辑 `MaiBot-Napcat-Adapter/config.toml`：
```toml
group_list = [你的群号]
private_list = [你的QQ号]
```

### 伊伊的白名单
编辑 `MaiBot-Napcat-Adapter-Yiyi/config.toml`：
```toml
group_list = [你的群号]
private_list = [你的QQ号]
```

## 第2步: 一键启动

### Windows
双击运行：
```
start_all_bots.bat
```

### Linux/Mac
```bash
./start_all_bots.sh
```

## 第3步: 验证

访问: http://localhost:8001/bot-management.html

应该看到两个机器人都在线。

---

## 如果出问题

1. **机器人不回复** → 检查白名单
2. **连接失败** → 检查Napcat是否启动
3. **端口冲突** → 修改config.toml端口

详细指南: `双机器人启动指南.md`
