# 伊伊启动快速参考

## 基本信息
- **QQ账号**: 1033245881
- **Bot ID**: yiyi_bot
- **Napcat端口**: 8097

## 快速启动（3步）

### 1. 配置白名单（首次必须）
编辑 `MaiBot-Napcat-Adapter-Yiyi/config.toml`：
```toml
group_list = [你的群号]
private_list = [你的QQ号]
```

### 2. 启动伊伊服务
```bash
# Windows
start_yiyi.bat

# Linux/Mac
./start_yiyi.sh
```

### 3. 启动MaiBot
```bash
cd MaiBot
python src/main.py
```

## 验证
访问: http://localhost:8001/bot-management.html
应该看到伊伊在线

## 测试
向伊伊发消息，她应该：
- 回复简短（"嗯""对""这个"）
- 不太主动
- 提到游戏会有反应

## 问题排查
1. 伊伊不回复 → 检查白名单
2. 端口冲突 → 修改config.toml端口
3. 连接失败 → 检查Napcat是否启动
